# UI rules — Крымская Кровельная

## 1. Базовый принцип
Интерфейс не должен выглядеть как универсальная стройка. Основа — светлый фон, графитовый текст, красный акцент, строгая сетка и сильная типографика.

## 2. Цветовая система
- Фон: `#F5F6F8`
- Белый: `#FFFFFF`
- Графитовый: `#1F2430`
- Тёмный графит: `#121722`
- Красный: `#D62828`
- Тёмный красный: `#B71C1C`
- Вспомогательный серый: `#8A9099`

Пропорция: 70 / 20 / 10.

## 3. Типографика
- Заголовки: Montserrat
- Основной текст и UI: Manrope
- Текст интерфейса без декоративности
- H1: 40–60 px desktop, 34–40 px tablet, 32–36 px mobile
- H2: 30–40 px desktop, 28–32 px mobile
- Основной текст: 16–18 px
- Цены и CTA: semi-bold / bold

## 4. Сетка и ритм
- Контейнер: max-width 1280 px
- Базовый горизонтальный padding: 16 px mobile, 24 px desktop
- Вертикальный шаг секций: 40 px mobile, 64 px desktop
- Карточки и формы с крупным радиусом: 24–32 px

## 5. Компонентные правила
### Кнопки
- Primary: красная заливка, белый текст, pill radius
- Secondary: белая, графитовая граница
- Никогда не использовать несколько равноправных красных кнопок подряд

### Карточки
- Белый фон
- Тонкая графитовая/чёрная граница на 10%
- Мягкая тень только на разделение слоёв
- Без иконографического шума и без крупных цветных плашек

### Формы
- Только нужные поля
- Лейблы над полями
- Один главный CTA
- Не использовать длинные чекбоксы, квизы и многоступенчатые формы на старте

### FAQ
- Аккордеон
- Плотный заголовок вопроса
- Спокойный текст ответа

### Кейсы
- Формат: проблема → решение → результат
- Карточка должна легко читаться без фото
- Фото блока подключаются позже, когда будет готов контент

### Секции цен
- Без декоративных таблиц
- Одна колонка услуги + одна колонка цены
- Цена не должна спорить с H2

### CTA
- Один главный тезис
- До двух кнопок
- Для mobile обязательна sticky-панель связи

## 6. Изображения
- Только реальные объекты, узлы, кровля, процесс
- Без стоковых улыбающихся строителей
- Без агрессивных градиентов и баннерной стилистики

## 7. Что не делать
- Не уводить сайт в пёстрый каталог стройуслуг
- Не перегружать фон текстурами
- Не использовать псевдо-industrial декор ради эффекта
- Не делать тяжёлые hero-секции с каруселями
- Не прятать ключевой текст в табах и аккордеонах выше первого экрана
