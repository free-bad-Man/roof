# Checklist первого деплоя на VPS

## До сервера
- [ ] Есть VPS с Ubuntu
- [ ] Домен направлен на IP сервера
- [ ] Есть SSH-доступ
- [ ] Ветка `main` содержит актуальный production-код
- [ ] Подготовлены production env-значения

## На сервере первый раз
- [ ] Установлены Docker, docker compose plugin, Nginx, Certbot
- [ ] Созданы директории `/opt/roof/app`, `/opt/roof/env`, `/opt/roof/backups/postgres`
- [ ] Репозиторий склонирован в `/opt/roof/app`
- [ ] Создан `/opt/roof/env/.env.db`
- [ ] Создан `/opt/roof/env/.env.production`
- [ ] Проверен `DATABASE_URL`
- [ ] Установлен Nginx-конфиг
- [ ] `nginx -t` проходит без ошибок
- [ ] Выпущен SSL-сертификат через Certbot

## Первый запуск
- [ ] Поднят контейнер `db`
- [ ] Выполнен `npx prisma migrate deploy`
- [ ] Поднят контейнер `app`
- [ ] `docker compose ps` показывает оба сервиса в `Up`
- [ ] Главная открывается по HTTPS
- [ ] Нет 502 от Nginx

## Проверка бизнеса после первого старта
- [ ] Открывается главная
- [ ] Открываются ключевые страницы услуг
- [ ] Работает форма заявки
- [ ] Заявка уходит в CRM
- [ ] Проверен один тестовый лид
- [ ] Проверены contact-кнопки: телефон / Telegram / мессенджеры

## После первого деплоя
- [ ] Зафиксирован рабочий commit production
- [ ] Сделан первый ручной backup БД
- [ ] Проверен `certbot renew --dry-run`
- [ ] Понятен порядок следующего релиза только из `main`
